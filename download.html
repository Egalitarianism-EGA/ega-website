Install the wallet and full node. That’s enough to join the network and mine.

Prebuilt packages

Latest: v0.2.0 MultiShield-4 (Linux x86_64) — no compiler required.

Download v0.2.0 (~120 MB) All releases
Tarball includes egad, ega-cli, ega-qt, easy-start / easy-mine (RandomX · Verthash · YespowerEGA · Scrypt), explorer & pool scripts. 
tar -xzf ega-0.2.0-linux-x86_64.tar.gz
cd ega-0.2.0-linux-x86_64
export PATH="$PWD/bin:$PATH"
bash scripts/easy-start.sh

Build from source (Linux)

git clone https://github.com/Egalitarianism-EGA/ega.git
cd ega
bash scripts/easy-install-linux.sh
bash scripts/easy-start.sh
bash scripts/easy-wallet.sh

Optional desktop menu icons: bash scripts/install-desktop-apps.sh
