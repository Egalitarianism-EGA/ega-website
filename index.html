What you can run

Real applications from the Core project — wallet, node, and mining.

Wallet Desktop app to receive and send EGA. Uses the same software as the full node. ega-qt
Node Keeps a copy of the chain and talks to peers. Leave it running when you can. egad
Mining CPU: RandomX / YespowerEGA. GPU: Verthash. Solo now; pool when operators deploy one. ega-cli · VerthashMiner

How a normal network works

Two (or two hundred) miners do not “share one PC” — they connect to the same chain.

1

Nodes sync

Peers connect on port 20201 and share blocks. Everyone works on the same tip.

2

Miners race

Each miner builds on the latest block. Whoever finds the next valid block wins that reward.

3

Optional pool

A pool splits rewards by work. Useful later; not required for a live chain.
