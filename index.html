Egalitarianism

A fair-launch coin you can actually run.

EGA is multi-algo proof-of-work: RandomX, Verthash, and YespowerEGA. No premine. Anyone with a normal PC, GPU, or low-power machine can participate.

Get the software Getting started

The three applications

Same project, three jobs — like a normal crypto stack.

Wallet

Receive and send EGA. Desktop GUI or command line.

ega-qt · ega-cli

Node

Holds the chain, talks to peers on port 20201.

egad

Miner

CPU or GPU. Solo, or many people on the same network.

CPU · GPU Verthash

Install on Linux

  1. Clone Egalitarianism-EGA/ega
  2. Run bash scripts/easy-install-linux.sh
  3. Optional menu apps: bash scripts/install-desktop-apps.sh (Wallet, Node, Miner)
  4. Start a node: bash scripts/easy-start.sh
  5. Get an address: bash scripts/easy-wallet.sh
  6. Mine: bash scripts/easy-mine.sh randomx

Full guide: getting-started.md

Solo mining vs many miners

Solo: you run a node and mine. If you find a block, that reward is yours.

Same network (normal blockchain): two or more people each run a node, connect with addnode=IP:20201, and mine. They share one chain tip. Whoever finds the next block keeps that block — same idea as Bitcoin.

Pool (optional later): splits rewards by work. Not required to launch.

Local test of two miners on one machine: bash scripts/two-node-mining-demo.sh
Friend setup: network-mining.md
