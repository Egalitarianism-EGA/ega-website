Mine EGA

Four algorithms. Solo on any full node, or pool for Verthash (GPU) and Scrypt.

Pick your hardware

RandomXModern CPUs — solo with the node
YespowerEGAWeaker CPUs / Pi — solo
VerthashGPU — pool stratum+tcp://105.225.100.58:3334
ScryptCPU/ASIC path — pool stratum+tcp://105.225.100.58:3336

1. Get the software

Download Linux v0.2.0 or build from source. Run a full node so you stay on the same chain.

egad -daemon
ega-cli getnewaddress

2. Join the network

# ~/.ega/ega.conf
addnode=105.225.100.58:20201

Seed is the community bootstrap node (home operator). Port 20201 must be reachable.

3. Solo mine (any algo)

ega-cli generatetoaddress 1 $(ega-cli getnewaddress) 10000000 randomx
# yespower-ega | verthash | scrypt

Or helpers in the release tarball: scripts/easy-mine.sh randomx

4. Pool mine (Verthash / Scrypt)

See Pool for stratum hosts and GPU flags.
