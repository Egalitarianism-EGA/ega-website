Mining

Three algorithms. Solo, multi-node, or pool.

Algorithms

RandomXModern CPUs — bash scripts/easy-mine.sh randomx
YespowerEGAWeaker CPUs / Pi — bash scripts/easy-mine.sh yespower-ega
VerthashGPUs — ega-verthash-miner

Solo

You run a node and mine to your wallet. Blocks you find are yours (after coinbase maturity).

bash scripts/easy-start.sh
bash scripts/easy-mine.sh randomx

Multi-node (same chain)

Two or more people each run egad, connect as peers, and mine. Same chain tip; each keeps the blocks they find.

Seed operator: leave egad running, port 20201 reachable.

Everyone else, in ~/.ega/ega.conf:

addnode=SEED_IP:20201

Local test on one PC:

bash scripts/two-node-mining-demo.sh

Pool (shared rewards)

A pool splits block rewards by work. See the pool page for setup and ports.

Continuous solo loop

ADDR=$(./src/ega-cli getnewaddress)
while true; do
  ./src/ega-cli generatetoaddress 1 "$ADDR" 10000000 randomx || sleep 1
done
