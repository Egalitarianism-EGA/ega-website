Get started

Install Core, run a node, open a wallet. Linux is ready today.

1. Install (Linux)

git clone https://github.com/Egalitarianism-EGA/ega.git
cd ega
bash scripts/easy-install-linux.sh

That builds egad, ega-cli, and related tools.

2. Desktop apps (optional)

bash scripts/install-desktop-apps.sh

Adds Egalitarianism Wallet, Node, and Miner to your app menu.

3. Start the node

bash scripts/easy-start.sh

Config: ~/.ega/ega.conf · P2P port 20201 · keep RPC local.

4. Wallet

bash scripts/easy-wallet.sh
# or GUI (if built with Qt):
./src/qt/ega-qt

Backup your wallet file after you fund it.

5. Mine

bash scripts/easy-mine.sh randomx

More options on the mining page.

Desktop wallet (Qt)

sudo apt install qtbase5-dev qttools5-dev-tools libqrencode-dev
./configure --with-gui=qt5 --enable-wallet --with-incompatible-bdb
make -j$(nproc)
./src/qt/ega-qt

Join someone else’s node

In ~/.ega/ega.conf:

addnode=THEIR_IP:20201
Windows build notes are in the Core repo under docs/ega/build-windows.md. macOS is not packaged yet.
