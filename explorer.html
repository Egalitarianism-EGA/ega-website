Block explorer

Browse height, blocks, and transactions against a live egad node.

Run it yourself (free)

Lightweight explorer ships with Core. Talks to your local node over RPC.

Public URL

Operators can expose it with Tailscale or a tunnel. When a public URL is published, it will be linked here.

Start explorer (Linux)

# node with tx index (once)
# add to ~/.ega/ega.conf:  txindex=1
# then reindex if the chain already exists without txindex:
#   egad -reindex-chainstate   # or full -reindex if needed

bash scripts/easy-start.sh
bash scripts/ega-explorer.sh
# open http://127.0.0.1:8088

What you get

  • Chain height, best hash, difficulty summary
  • Latest blocks
  • Lookup block by height or hash
  • Lookup transaction by txid (needs txindex=1)
This site is static (GitHub Pages). The explorer is a small app you run next to egad — same idea as most early coin explorers before someone hosts a public one.
