Mining pool

Shared rewards when several miners want payouts proportional to work. Not required for a live multi-node network — only if you want a stratum pool.

Suggested ports

RandomXstratum port 3333
Verthashstratum port 3334
YespowerEGAstratum port 3335
Pool API4000 (Miningcore default in our example)

Run on your PC (Docker)

From Core (after egad is running with wallet + RPC):

cd ega-homelab   # or scripts path in Core
bash scripts/start-pool-stack.sh

Full configs: ega-miningcore · home lab: ega-homelab

Miningcore must be built/configured with coin profiles that match EGA’s RPC and algos. RandomX is the most common first pool port. Verthash and YespowerEGA need matching miner + pool algorithm support — start with RandomX if you are ironing things out.

Without a pool

Friends can still mine the same chain by connecting nodes (addnode=IP:20201) and solo-mining. See Mining.
