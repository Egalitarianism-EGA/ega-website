Mining pool

How a public stratum pool will look once operators publish a host. There is no public EGA pool hostname yet — ports below are the planned layout.

Status

Not live as a public service. Solo mining and friend-to-friend nodes work today (see Mine). Operators can run Miningcore locally (Verthash on :3334 is the path that does real share verification today).

Planned ports (when a host is published)

RandomXstratum+tcp://POOL_HOST:3333
Verthashstratum+tcp://POOL_HOST:3334
YespowerEGAstratum+tcp://POOL_HOST:3335
Scryptstratum+tcp://POOL_HOST:3336
UsernameYour EGA wallet address
Passwordx (or worker name)
Replace POOL_HOST only after an operator publishes one. Until then these are placeholders — not fake live endpoints.
